게임

홀카드hole cards

각자에게 비공개로 나눠지는 2장의 카드.

보드board / community cards

테이블 중앙에 공개되는 공유 카드 5장(플랍3+턴1+리버1).

플랍 / 턴 / 리버flop / turn / river

공유 카드가 공개되는 순서. 플랍 3장, 턴 1장, 리버 1장.

블라인드blind (SB/BB)

매 핸드 강제로 내는 베팅. 스몰 블라인드 0.5, 빅 블라인드 1.

쇼다운showdown

마지막까지 남은 플레이어들이 핸드를 공개해 승자를 가리는 것.

팟pot

한 핸드에서 모인 칩의 총합. 승자가 가져감.

스택 / 이펙티브 스택stack / effective stack

가진 칩의 양. 이펙티브 스택은 맞붙은 두 명 중 더 적은 스택(실제로 걸 수 있는 한도).

bb (빅블라인드 단위)big blinds

스택·베팅을 빅 블라인드 개수로 표현. 예: 20bb = 빅블라인드의 20배.

포지션

UTG / HJ / CO / BTNpositions

행동 순서대로의 자리. UTG(가장 먼저)~BTN(버튼, 가장 늦게). 늦을수록 유리.

버튼 (BTN)button

딜러 위치. 프리플랍 마지막에 가깝고 포스트플랍 항상 마지막 행동 → 가장 유리.

인포지션 / 아웃오브포지션IP / OOP

포스트플랍에서 상대보다 나중에(IP) 또는 먼저(OOP) 행동하는지. IP가 유리.

스틸steal

후순위에서 블라인드를 노리고 넓게 오픈해 거저 가져오려는 레이즈.

액션

폴드 / 체크 / 콜fold / check / call

포기 / 베팅 없이 넘김 / 상대 베팅만큼 맞춤.

벳 / 레이즈bet / raise

먼저 칩을 거는 것 / 상대 베팅 위에 더 올리는 것.

오픈 / RFIopen / raise first in

아무도 안 들어왔을 때 처음으로 레이즈하는 것. 이니셔티브를 가짐.

3벳 / 4벳 / 5벳3-bet / 4-bet / 5-bet

리레이즈의 단계. 오픈이 2벳, 그 위가 3벳, 또 위가 4벳…(보통 5벳은 올인).

올인 / 잼all-in / jam / shove

스택 전부를 거는 것. 숏스택에서 폴드이쿼티+에쿼티를 한 번에 챙김.

리슈브reshove / re-jam

상대 오픈에 대해 올인으로 되받는 것.

림프limp

레이즈 없이 빅블라인드만 콜하며 들어가는 것. HU 외엔 보통 약함.

아이소레이즈isolation raise

림퍼를 상대로 1:1을 만들기 위한 레이즈.

c-벳continuation bet

프리플랍 레이저가 플랍에서 이어서 하는 베팅.

핸드

페어 / 수딧 / 오프숏pair / suited / offsuit

같은 숫자 / 같은 무늬 두 장 / 다른 무늬 두 장. 수딧이 오프숏보다 약간 강함.

커넥터connector

숫자가 이어지는 두 장(예: 87, JT). 스트레이트 잠재력.

브로드웨이broadway

T 이상(T,J,Q,K,A)의 높은 카드.

키커kicker

같은 조합일 때 승부를 가르는 곁다리 카드.

콤보combo

무늬까지 구분한 구체적 조합 수. 페어 6, 수딧 4, 오프숏 12.

블로커blocker

내가 든 카드가 상대의 특정 핸드 조합을 줄여주는 효과(예: A를 들면 상대 AA/AK 감소).

개념

에쿼티equity

지금 끝까지 갔을 때 이길 확률(타이는 절반).

팟 오즈pot odds

콜 비용 대비 받는 가격. 필요 에쿼티 = 콜 ÷ (콜 후 전체 팟).

EV (기대값)expected value

각 결과의 (확률×값)의 합. 모든 결정의 기준.

EV 손실EV loss

최적 액션 EV − 내 선택 EV. 0이면 완벽한 플레이.

임플라이드 오즈implied odds

맞췄을 때 추가로 벌 미래 베팅까지 감안한 가격.

폴드 이쿼티fold equity

상대가 접을 확률에서 오는 가치. 베팅/올인의 핵심 동력.

레인지range

어떤 액션을 하는 가능한 핸드들의 집합. 상대를 한 핸드가 아닌 레인지로 봄.

밸류벳 / 블러프value bet / bluff

콜 받아 이기려는 베팅 / 폴드 받으려는 베팅. 둘을 섞어야 균형.

세미블러프semi-bluff

지금은 약하지만 드로우가 있어 맞으면 강해지는 핸드로 하는 블러프.

폴라라이즈 / 머지드polarized / merged

레인지가 강함+블러프로 양극화 / 중간 강도 위주로 뭉쳐있음.

MDFminimum defense frequency

베팅에 착취당하지 않으려 지켜야 하는 최소 비율 = 1 − α.

α (알파)alpha

벳 ÷ (벳 + 팟). 상대를 무차별하게 만드는 블러프 비율의 기준.

GTO

GTOgame theory optimal

상대가 뭘 하든 일방적으로 착취당하지 않는 전략. = 내쉬 균형.

내쉬 균형Nash equilibrium

아무도 혼자 전략을 바꿔서 더 못 버는 상태.

Exploitabilityexploitability

그 전략을 상대가 최대로 노렸을 때 잃는 양. 0이면 완벽한 균형.

익스플로잇exploit

상대의 약점에 맞춰 GTO에서 벗어나 더 버는 것.

혼합 전략 / 무차별mixed strategy / indifference

같은 핸드를 여러 액션으로 섞음. 액션 EV가 같아질(무차별) 때 최적.

솔버 / CFRsolver / CFR

GTO 해를 계산하는 프로그램 / 그 대표 알고리즘(반사실적 후회 최소화).

도달 확률reach

특정 핸드가 그 스팟까지 올 확률. 낮으면 그 스팟의 레인지에 거의 없음.

토너먼트

ICMindependent chip model

토너먼트에서 칩을 상금 기대값으로 환산하는 모델. 칩 가치는 비선형.

cEV / $EVchip EV / dollar EV

칩 기준 기대값 / 상금 기준 기대값. 토너먼트 후반엔 $EV로 판단.

버블 / 페이점프bubble / pay jump

상금권 직전 / 상금 액수가 바뀌는 경계. 근처에선 더 타이트하게.

숏 / 미들 / 빅 스택short / mid / big stack

상대적 스택 크기. 빅은 압박, 미들은 몸 사림, 숏은 먼저 움직임.

보드

드라이 / 웻 보드dry / wet board

드로우·연결이 적은 / 많은 플랍. 전략(빈도·사이즈)이 달라짐.

페어드 / 모노톤paired / monotone

같은 숫자가 포함된 보드 / 같은 무늬 3장인 보드.

텍스처board texture

플랍의 성격(드라이/웻/페어드 등). c-벳 전략의 핵심 변수.

레인지 우위range advantage

특정 보드에 어느 쪽 레인지가 더 잘 맞는지. 베팅 빈도·사이즈를 좌우.

SPRstack-to-pot ratio

남은 스택 ÷ 팟. 낮으면 커밋되기 쉽고, 높으면 포지션·기술이 더 중요.
